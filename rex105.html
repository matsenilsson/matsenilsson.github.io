Notes REX105

Today just a little more on functions, especially on those that take functions as arguments (so called functionals). There are many useful functionals in R, including functions in the apply family. Below I will just give a few examples. (Google ‘R apply family’ to find numerous examples and tutorials.)

   

Functions as arguments

Some functions accept functions as arguments. We have already used some of these, for example apply() that applies a function to the rows or columns of a homogeneous data frame (a data frame with only one type of elements, i.e., a matrix). Here is another example:

# Simulate responses on 10 questionnaire items, each scored 1, 2, ..., 5
set.seed(123)
d <- data.frame(replicate(10, sample(1:5, 100, replace = TRUE)))

# Means across rows
meanrows <- apply(d, MARGIN = 1, FUN = mean) 
summary(meanrows)
##    Min. 1st Qu.  Median    Mean 3rd Qu.    Max. 
##   1.700   2.675   3.000   2.989   3.300   4.100
# Means across columns
meancols <- apply(d, MARGIN = 2, FUN = mean) 
meancols
##   X1   X2   X3   X4   X5   X6   X7   X8   X9  X10 
## 3.00 3.07 2.92 2.94 2.93 3.09 3.01 3.00 2.92 3.01

   

Anonomous functions

Anonymous functions are functions without name. Typically they are performing simple tasks that can be defined in a single row of code. Anonymous functions are frequently used in functionals. Here is an example using the data generated above. The anonymous function function(x) sum((x - mean(x)^2)) calculates the sum of squares of its input, which in this case is the columns of d (as I set MARGINS = 2):

# Calculate sum of squares across columns of d
sscols <- apply(d, MARGIN = 2, FUN = function(x) sum((x - mean(x))^2)) 
sscols
##     X1     X2     X3     X4     X5     X6     X7     X8     X9    X10 
## 190.00 168.51 205.36 199.64 202.51 212.19 210.99 202.00 191.36 198.99

   

lapply

There are a whole set of functions in the apply family. One of the more useful is lapply() that does something on each element of the input and returns the result in a list. Here is an example on how to use it on a data frame containing columns of different types.

# iris is a datset preloaded in R. This checks the class of each column
lapply(iris, FUN = class)
## $Sepal.Length
## [1] "numeric"
## 
## $Sepal.Width
## [1] "numeric"
## 
## $Petal.Length
## [1] "numeric"
## 
## $Petal.Width
## [1] "numeric"
## 
## $Species
## [1] "factor"
# This counts missing values in each column
lapply(iris, function(x) sum(is.na(x)))
## $Sepal.Length
## [1] 0
## 
## $Sepal.Width
## [1] 0
## 
## $Petal.Length
## [1] 0
## 
## $Petal.Width
## [1] 0
## 
## $Species
## [1] 0

The return is a list. If you prefer to have it as a vector, try sapply() (s for simplify)

sapply(iris, class)
## Sepal.Length  Sepal.Width Petal.Length  Petal.Width      Species 
##    "numeric"    "numeric"    "numeric"    "numeric"     "factor"
sapply(iris, function(x) sum(is.na(x)))
## Sepal.Length  Sepal.Width Petal.Length  Petal.Width      Species 
##            0            0            0            0            0

   

tapply

As a final example of functionals, let’s have a look at the function tapply(). This function allows you to apply a function on subsets of your data. Here is a simple example calculating the median sepal length for each species in the iris dataset.

tapply(iris$Sepal.Length, INDEX = list(iris$Species), FUN = median, na.rm = TRUE)
##     setosa versicolor  virginica 
##        5.0        5.9        6.5

Note that the second argument (INDEX) has to be a list. It may be a list of several variables by which to split the analysis:

# Create a new dataframe from the iris dataset
d <- iris
# Add dichotomous variable, distinguishing narrow and wide sepals.
d$sepal_width_dicho <- factor(1 * (d$Sepal.Width > median(d$Sepal.Width)), 
                              levels = c(0, 1), labels = c('narrow', 'wide'))

# Calculate median sepal length for each species and sepal width category
tapply(d$Sepal.Length, INDEX = list(d$Species, d$sepal_width_dicho), FUN = median)
##            narrow wide
## setosa       4.65 5.10
## versicolor   5.75 6.55
## virginica    6.40 6.70
